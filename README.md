#### 关于

- 本样式库中的样式类名大多都是分离封装的，且类名呈现方式大多是根据**原型样式全名的缩写**或发音缩写而设置的，便于快速上手
- 本样式库的优点：分离式封装的类名具有极高的组合灵活性、代码可读性和后期维护性
- 而缺点也很显而易见：某些标签可能为满足需求方特殊的样式要求，会需要写很长一串类名
